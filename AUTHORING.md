# איך כותבים פוסט בבלוג

הבלוג בנוי כך שכתיבת פוסט היא פשוטה ככל האפשר. אתם לא נוגעים ב-HTML — רק
כותבים ב-Markdown, וסקריפט בנייה מייצר את כל השאר.

## הדרך המהירה (מהדפדפן)

1. פתחו את `admin.html` בדפדפן (בכתובת `https://galaviel.com/admin.html`).
2. הזינו את הסיסמה (ברירת מחדל: `gal-blog` — **כדאי להחליף**, ראו למטה).
3. מלאו כותרת, תגיות, תקציר וגוף הפוסט. יש תצוגה מקדימה חיה מימין.
4. לחצו **הורד .md** ושמרו את הקובץ בתיקיית `content/`.
5. עשו commit ו-push. ה-GitHub Action יבנה את האתר אוטומטית.

## הדרך הידנית (מהמחשב)

1. צרו קובץ חדש `content/<slug>.md` (ה-slug הופך לשם העמוד, למשל
   `content/my-post.md` → `posts/my-post.html`).
2. בראש הקובץ הוסיפו בלוק frontmatter:

   ```
   ---
   title: כותרת הפוסט
   date: 2026-06-13
   tags: פיתוח, iOS
   excerpt: משפט קצר שמתאר את הפוסט (מופיע ברשימה ובתוצאות חיפוש).
   readingTime: 4
   ---
   ```

   - `title`, `date` (בפורמט `YYYY-MM-DD`) ו-`excerpt` הם **חובה**.
   - `tags` ו-`readingTime` אופציונליים. אם לא תציינו `readingTime`, הוא יחושב
     אוטומטית לפי אורך הטקסט.

3. מתחת ל-frontmatter כתבו את גוף הפוסט ב-Markdown:

   | תחביר | תוצאה |
   |-------|-------|
   | `## כותרת` | כותרת משנה |
   | `**טקסט**` | מודגש |
   | `*טקסט*` | נטוי |
   | `` `קוד` `` | קוד מוטבע |
   | ``` ```code``` ``` | בלוק קוד |
   | `- פריט` | רשימה |
   | `1. פריט` | רשימה ממוספרת |
   | `> ציטוט` | ציטוט |
   | `[טקסט](url)` | קישור |

4. הריצו את הבנייה:

   ```
   node build.js
   ```

5. עשו commit ו-push.

## מה נוצר אוטומטית

הסקריפט `build.js` מייצר את כל אלה — **אל תערכו אותם ידנית**:

- `posts/<slug>.html` — עמוד הפוסט המלא, כולל תגיות meta ו-JSON-LD ל-SEO.
- `posts.js` — האינדקס שמזין את החיפוש והסינון ב-`blog.html`.
- `tags/<slug>.html` — עמוד עם URL קבוע לכל תגית.
- `sitemap.xml` ו-`rss.xml`.

## להחלפת סיסמת העורך

הסיסמה ב-`admin.html` היא **מנעול נוחות בלבד**, לא אבטחה אמיתית (הקוד גלוי
לכולם). ההגנה האמיתית היא שהאתר סטטי — פרסום דורש commit ל-git.

כדי להחליף את הסיסמה, חשבו את ה-SHA-256 של הסיסמה החדשה:

```
node -e "console.log(require('crypto').createHash('sha256').update('הסיסמה-שלי').digest('hex'))"
```

והדביקו את התוצאה ב-`admin.html` במשתנה `PASS_HASH`.
