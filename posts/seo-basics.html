SEO בסיסי לאתר סטטי

#SEO #ווב #פיתוח

לאתר סטטי אין צורך בתוספי SEO כבדים. כמה החלטות נכונות ב-HTML עצמו עושות את רוב העבודה. הנה הרשימה שאני עובר עליה לכל עמוד.

1. כותרת ותיאור

כל עמוד צריך <title> ייחודי ו-meta description קצר (כ-150 תווים) שמתאר במדויק את התוכן. אלה מה שגוגל מציג בתוצאות החיפוש, אז כדאי שיהיו מזמינים.

2. Open Graph ו-Twitter Cards

כשמישהו משתף קישור בוואטסאפ, פייסבוק או X — תגיות og:title, og:description ו-og:image קובעות איך התצוגה המקדימה תיראה. בלעדיהן הקישור נראה עירום.

3. נתונים מובנים (JSON-LD)

בלוק application/ld+json עם סכמת BlogPosting עוזר למנועי חיפוש להבין שמדובר במאמר, מי כתב אותו ומתי. זה מה שמאפשר תוצאות עשירות (rich results). 

{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "כותרת הפוסט",
  "datePublished": "2026-05-10",
  "author": { "@type": "Person", "name": "Gal Aviel" }
}

4. Sitemap ו-robots.txt

קובץ sitemap.xml מפרט את כל העמודים שלכם כדי שגוגל ימצא אותם מהר. robots.txt מצביע אל ה-sitemap ומגדיר מה מותר לסרוק. שניהם קבצים פשוטים בשורש האתר.

5. RSS — עדיין רלוונטי

הזנת RSS מאפשרת לקוראים לעקוב אחרי הבלוג בקוראי תוכן. זה גם ערוץ הפצה שאתם שולטים בו במלואו, בלי תלות באלגוריתם של אף רשת חברתית.

הכלל הפשוט: כתבו תוכן טוב, ותנו למנועי החיפוש את ההקשר המינימלי שהם צריכים כדי להבין אותו. כל השאר הוא ליטוש.

מה פחות חשוב

  • צפיפות מילות מפתח — מיושן, ולפעמים מזיק.
  • תגיות meta keywords — גוגל מתעלם מהן מזמן.
  • טריקים לקיצור דרך — מסוכנים יותר משמועילים.
